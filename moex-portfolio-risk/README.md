# Оценка риска инвестиционного портфеля методами классического ML и Deep Learning (PyTorch)

Реализация сквозного пайплайна: от автоматизированного сбора реальных рыночных данных с Московской Биржи до генерации синтетического датасета и обучения нейросети на PyTorch для многоклассовой классификации уровней риска портфелей.

## Описание проекта
В рамках проекта решена задача оценки и классификации инвестиционных рисков для диверсифицированных портфелей акций. Проект демонстрирует навыки работы с финансовыми временными рядами, математического моделирования (метод Монте-Карло) и построения кастомных нейросетевых архитектур в экосистеме PyTorch.

## Стек технологий
* **Язык:** Python
* **Сбор данных:** `apimoex` (Импорт данных напрямую через Moex API)
* **Анализ данных и вычисления:** Pandas, NumPy, Scikit-learn
* **Глубокое обучение:** PyTorch (Кастомные Dataset/DataLoader, nn.Sequential, Adam)
* **Визуализация:** Matplotlib, Seaborn

## Этапы реализации
1. **Сбор данных (Data Engineering):** Выгрузка исторических данных по закрытиям акций первого и второго эшелонов Московской Биржи за последние 5 лет.
2. **Математическое моделирование (Monte Carlo Simulation):**
   * Расчет матрицы ковариации и исторической доходности активов.
   * Симуляция **50 000 случайных портфелей** с генерацией случайных весов долей активов.
   * Расчет ожидаемой доходности, волатильности (риска) и коэффициента Шарпа для каждого сгенерированного портфеля.
3. **Разметка данных:** Разделение портфелей на 3 класса риска (Низкий / Средний / Высокий) на основе квантилей полученного распределения волатильности.
4. **Обучение модели (Deep Learning Pipeline):**
   * Создание кастомного пайплайна на PyTorch для нормализации признаков (весов активов).
   * Проектирование полносвязной глубокой нейросети (MLP) с функциями активации ReLU и слоями Dropout для предотвращения переобучения.
   * Оптимизация кросс-энтропийной функции потерь (CrossEntropyLoss).

## Результаты
* Построена эффективная модель классификации рисков на основе долей активов, достигшая высокой сходимости на тестовой выборке.
* Визуализирована граница эффективности Марковица (Efficient Frontier) на основе симулированных Монте-Карло портфелей с цветовым кодированием по классам риска.
