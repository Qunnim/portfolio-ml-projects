Добро пожаловать в мой репозиторий с проектами в области машинного обучения, анализа данных и глубокого обучения. Здесь собраны практические кейсы, реализованные мной в ходе обучения в Финансовом университете.

## Навигация по проектам

### 1. [Оценка риска инвестиционного портфеля на данных Мосбиржи](./moex-portfolio-risk/)
* **Задача:** Многоклассовая классификация инвестиционных рисков диверсифицированных портфелей.
* **Стек:** Python, PyTorch, Moex API (`apimoex`), Pandas, Monte Carlo Simulation.
* **Краткий результат:** Сгенерирован синтетический датасет из 50 000 портфелей на основе 5-летних исторических данных акций. Обучена полносвязная нейросеть для классификации уровней риска на основе весов активов и ковариационной матрицы.
* ** [Перейти к проекту и README](./moex-portfolio-risk/)**

### 2. [Бенчмаркинг алгоритмов ML и DL на датасете Fashion-MNIST](./fashion-mnist-classification/)
* **Задача:** Сравнительный анализ 10 различных подходов к классификации изображений одежды.
* **Стек:** Python, TensorFlow/Keras, Scikit-learn (SVM, Random Forest, Градиентный бустинг), PCA, t-SNE.
* **Краткий результат:** Спроектирована глубокая сверточная нейросеть (DeepCNN), достигшая точности **Accuracy: ~93.2%** на тестовой выборке. Проведен качественный анализ ошибок классификации с помощью матрицы несоответствий.
* ** [Перейти к проекту и README](./fashion-mnist-classification/)**
