# Сравнительный анализ алгоритмов Machine Learning и Deep Learning на датасете Fashion-MNIST

Комплексное исследование и бенчмаркинг 10 различных подходов (от классических линейных моделей до глубоких сверточных нейросетей) в задаче компьютерного зрения (Computer Vision).

## Описание проекта
Цель работы — изучить эволюцию подходов к классификации изображений и на практике доказать превосходство сверточных архитектур над классическим машинным обучением при работе с пространственными структурами данных. Набор данных содержит 70 000 нормализованных изображений элементов одежды 10 классов в градациях серого (размер 28x28).

## Стек технологий
* **Библиотеки ML:** Scikit-learn (SVM, Random Forest, HistGradientBoosting, Logistic Regression, Naive Bayes)
* **Библиотеки DL:** TensorFlow / Keras
* **Снижение размерности:** PCA, t-SNE, UMAP
* **Анализ и визуализация:** NumPy, Pandas, Matplotlib, Seaborn

## Проведенные эксперименты и архитектуры
В ходе исследования были обучены и протестированы:
1. **Классический ML (с предварительным PCA-100 снижением размерности):** Logistic Regression, Random Forest, SVM (RBF kernel), Naive Bayes и др.
2. **Полносвязная нейросеть (MLP):** Базовая архитектура с плоским входным слоем, скрытыми слоями Dense и Dropout.
3. **Свёрточные нейросети (CNN):** 
   * `SimpleCNN` (Базовая сверточная архитектура).
   * `DeepCNN` (Глубокая сверточная сеть с блоками Conv2D -> MaxPooling2D -> Dropout).
   * Режим со свёрточной архитектурой + **аугментация данных** для оценки устойчивости к оверфиттингу.

## Итоговый бенчмаркинг моделей (Диапазон Test Accuracy)

| Подход | Test Accuracy | Ключевые наблюдения |
| :--- | :---: | :--- |
| **Классический ML (PCA-100)** | ~0.78 – 0.89 | **SVM RBF** показал лучший результат (0.8922) среди классических алгоритмов. Naive Bayes — наихудший. |
| **MLP (Полносвязная сеть)** | ~0.8935 | Сопоставима с SVM, но уступает CNN из-за отсутствия пространственной инвариантности. |
| **CNN (Свёрточные сети)** | **~0.931 – 0.932** | **Лучший результат (DeepCNN: 0.9322).** Значительное превосходство благодаря автоматическому извлечению локальных признаков. |
| **CNN + Аугментация** | ~0.8936 | Высокая устойчивость к переобучению, потенциал генерализации при увеличении числа эпох обучения. |

## Метрики и Анализ ошибок
В проекте реализовано построение детальной матрицы ошибок (`Confusion Matrix`). Проведен качественный анализ пересечений классов, показавший, что модель чаще всего колеблется между визуально похожими паттернами: *T-shirt/top* vs *Shirt* и *Coat* vs *Pullover*, что открывает направления для дальнейшей оптимизации архитектуры.
